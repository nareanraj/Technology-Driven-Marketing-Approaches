# Technology-Driven Marketing Approaches

A professional, corporate-style website built with HTML and CSS showcasing modern marketing technologies.

## Features
- **Clean Corporate Design**: Dark navy blue theme with modern typography.
- **Responsive Layout**: Mobile-friendly navigation and grid systems.
- **Interactive Elements**: Smooth scrolling, hover effects, and transitions.
- **Modern Tech Stack**: Semantic HTML5 and Vanilla CSS3.

## Project Structure
- `index.html`: Main website structure.
- `style.css`: Professional styling and animations.
- `planning.md`: Day 1 planning and design documentation.
- `assets/`: Image assets including the AI-generated hero background.

## How to Run
Simply open `index.html` in any modern web browser.
